This guide walks you through setting up my-company-llm on your local machine.

## Prerequisites

- **Python 3.12+** — for the FastAPI backend
- **Node.js 18+** — for the Next.js frontend
- **OpenAI API key** — used for GPT-4o agents and embeddings
- **Anthropic API key** — used for Claude-based creative agents

## Installation

### 1. Clone and Run

```bash
git clone https://github.com/taha-can/my-company-llm.git
cd my-company-llm
cp .env.example .env
# Edit .env with your API keys
```

This will set up the environment, install dependencies, and start both the backend and frontend.

The API will be available at `http://localhost:8000` and the frontend at [http://localhost:3000](http://localhost:3000).

## First Run

When you first open the app, you'll see the **Welcome Screen**. Enter your company details:

1. **Company Name** — the name of your organization
2. **Description** — what your company does
3. **Industry** — your industry sector
4. **Brand Voice** — how your company communicates

This information is shared with all AI agents as context, helping them align their responses with your brand.
